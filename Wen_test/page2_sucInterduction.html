虹之玉

虹之玉（學名：Sedum x rubrotinctum Clausen）是景天科景天屬的多肉植物，原生於墨西哥。它是一種多年生肉質草本植物，因其美麗的外觀和易於繁殖而受到廣泛喜愛。

About 虹之玉+

    外觀特徵

  • 葉片肉質，呈長橢圓形，約1厘米長，通常互生排列。
  • 在充足陽光的環境下，葉片會逐漸轉紅，形成紅綠相間的迷人色彩。
  • 花朵為黃色星形，主要在秋冬季節或強光照射下開放，此時葉片部分或全部可能轉為鮮紅色。

    • 生長環境

  • 喜歡溫暖、光照充足的環境，生長適溫為10~28℃。
  • 夏季應避免長時間暴曬，可移至散光處以防止葉片過度曬傷。
  • 耐旱性強，春、秋季節生長期每週澆水一次即可，夏季高溫時減少澆水，宜多噴水霧以增加空氣濕度。
  • 冬季需控制澆水量，避免過濕。

    • 栽培技巧

  • 虹之玉容易群生，適合盆栽或組合盆景使用。
  • 繁殖主要以葉片插芽方式進行，繁殖速度快，成活率高。


桃之卵

桃之卵（學名：Echeveria 'Topsy Turvy'）是景天科的一種多肉植物，以其特殊的葉片形狀和美麗的外觀而受到人們喜愛。以下是桃之卵的一些主要特徵和栽培注意事項：

About 桃之卵+

    外觀特徵

  • 桃之卵的葉片呈卵形，但與一般的多肉植物不同，其葉片邊緣向上捲曲，呈現出獨特的反捲狀態，有時會呈現深紫色的邊緣。
  • 葉片表面光滑，質感肉厚，呈灰綠色或藍綠色，較少見的品種可能會有淡紅色的葉片。

    • 生長環境

  • 桃之卵喜歡溫暖、乾燥且充足的陽光環境。適宜的生長溫度為攝氏15至25度。
  • 在生長季節，可以放置在充足陽光下，如室外陽台或窗戶邊。
  • 冬季需保持較低的溫度，避免過度澆水，防止根部腐爛。
  • 如果長時間缺乏陽光，葉片可能會變得扁平，失去其獨特的捲曲形狀，因此需要定期移動位置以確保光照均勻。

    • 栽培技巧

  • 為了促進健康生長，建議使用排水良好的多肉植物專用土壤，或者自行配製沙質和礫石的混合土壤。
  • 澆水要適度，以保持土壤微濕即可，避免積水和過濕，特別是在冬季。
  • 桃之卵屬於緩慢生長的多肉植物，不需要頻繁施肥，每隔兩個月施一次多肉植物專用肥料即可。


玉露

玉露（學名：Crassula ovata），又稱為翡翠植物或者金錢樹，是一種常見且受歡迎的多肉植物，特别是在室内養護方面。以下是玉露的介绍：

About 玉露+

    外觀特徵

  • 葉片特徵：玉露的葉片呈橢圓形或卵形，肉質厚實，通常呈現鮮綠色，有時帶有淡黃或紅色的邊緣。
  • 植株形態：通常生長為小型灌木狀，枝條粗壯，有時會逐漸變粗成為樹幹狀。

    • 生長環境

  • 光照需求：喜歡充足的陽光，但也能適應較弱的光線環境。在室內，可以選擇放置在明亮的窗台上。
  • 溫度要求：喜歡溫暖環境，理想生長溫度為攝氏15-24度。
  • 水分管理：耐旱，喜歡乾燥的環境，避免過多的澆水。春夏季節生長期間保持土壤微濕即可，冬季可減少澆水頻率。

    • 栽培技巧

  • 土壤選擇：使用排水良好的多肉植物專用土壤，或自製沙質土壤與少量腐葉土的混合物。
  • 施肥：春夏季節每月施一次稀釋的多肉植物專用肥料，秋冬季節減少或停止施肥。
  • 修剪與整形：可以根據需要修剪，促進植株形態的整齊。


大雪蓮

大雪蓮（學名：Sedum spectabile），又名秋英或秋英多肉，是景天科大雪蓮屬植物的一種，身上會鋪上一層厚厚的白粉，以下是大雪蓮的介紹：

About 大雪蓮+

    外觀特徵

  • 植株形態：大雪蓮是多年生草本植物，株高約30至60厘米，植株呈現直立或稍彎曲的姿態。
  • 葉片：葉片肉質，呈卵形或橢圓形，對生或互生排列，通常呈青綠色。
  • 花朵：花朵呈圓錐狀花序，花瓣五瓣，色彩豐富，主要有粉紅、白色或紅色等不同變種。

    • 生長環境

  • 光照需求：喜歡充足的陽光，生長最佳的環境是全日照或半日照，適合種植在陽光充足的地方。
  • 耐寒性：耐寒，適應性強，可以耐受寒冷的冬季天氣，不過極端寒冷的地區可能需要適當的保護。，葉片在冬季可能會因為低溫而轉紅或變色，增添花圃的冬季景觀。
  • 土壤要求：喜歡排水良好的土壤，可以在沙質或砂質壤土中良好生長，避免積水。

    • 栽培技巧

  • 澆水管理：生長期間保持土壤微濕，但不要過濕，特別是在冬季要減少澆水頻率。
  • 施肥：生長期間每隔一個月施一次磷鉀肥料，有助於促進花芽的形成和生長。
  • 繁殖方法：大雪蓮可以通過分株、扦插或種子繁殖，分株是最常見和快速的繁殖方式。


黃麗

黃麗（學名：Echeveria 'Huang Lih'）是一種多肉植物，屬於景天科（Crassulaceae）的一員。以下是關於黃麗的介紹：

About 黃麗+

    外觀特徵

  • 黃麗擁有厚實多汁的葉片，典型景天科植物的特徵。葉片呈美麗的綠色，有時帶有些許紅色或粉紅色的邊緣，因此深受多肉植物愛好者喜愛。

    • 生長環境

  • 黃麗耐干旱，適合生長在陽光充足、通風良好的環境中。它們喜歡乾燥的生長環境，因此需注意避免過度澆水，以免導致根部腐爛。

    • 栽培技巧

  • 黃麗可透過葉片扦插或分株繁殖。扦插時，將健康的葉片摘下，放置在乾燥通風良好的環境中，等待新根生長。分株則是將成長在一起的多肉植物分開，每個分株都具有獨立的生長能力。
  • 養護上，定期施肥可保持其健康生長，適當的陽光照射和適度的水分管理是保持健康的關鍵。


熊童子

熊童子（學名：Kalanchoe tomentosa），俗稱熊爪或熊童子，是一種景天科多肉植物，原產於馬達加斯加。這種植物以其獨特的外觀和容易栽培而廣受喜愛。以下是熊童子的詳細介紹：

About 熊童子+

    外觀特徵

  • 葉片特徵：熊童子的葉片厚實，呈卵形或倒卵形，邊緣有鋸齒狀突起，整體呈現灰綠色或灰白色，表面覆蓋著細小的白色絨毛，這些絨毛使得葉片觸感柔軟。
  • 生長形態：熊童子通常生長為小型灌木狀，高度約15至30厘米，枝幹直立或稍微弓曲，支持著密集的葉片。

    • 生長環境

  • 光照需求：喜歡充足的明亮光線，適合種植在室內明亮的位置，如窗邊或陽台。
  • 溫度要求：熊童子對溫度要求不苛刻，適應性強，理想生長溫度在攝氏15至25度之間。
  • 澆水管理：耐旱，植株在生長期間保持土壤微濕即可，冬季減少澆水頻率以防止過濕。

    • 栽培技巧

  • 土壤選擇：使用排水良好的多肉植物專用土壤或沙質壤土混合物，以確保良好的通氣性和排水性。
  • 施肥：春夏季節每隔兩個月施一次稀釋的多肉植物專用肥料，有助於促進植株生長和健康。
  • 繁殖方法：熊童子可以通過葉片插芽或枝條扦插繁殖，方法簡單且成功率高。
